# 985 Preflight Before 6551 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refresh 985 once before any 6551 point-spending fallback so slow 985 ingestion does not burn points.

**Architecture:** Keep 985 as the primary source and 6551 as fallback. When hybrid backfill reaches an `enrich` gate, run one 985 catchup for the batch/tick, re-check local `x_feed`, then only call 6551 for tweets still missing.

**Tech Stack:** Next.js route handlers, Node worker scripts, local SQLite store, existing 985 REST catchup utilities.
